## vivado开发环境

vivado2017.1版本（鉴于安装包比较大，不在此处提供连接，同学直接相互拷贝，辛成鑫U盘中有备份）

1.本安装包支持win7、10、Ubantu等系统（64位）。

2.解压后，在点击xsetup即可进入安装向导界面。

点击next

![](/assets/import.png)

点击所有Agree



![](/assets/import1.png)

进入版本选择

![](/assets/import2.png)



若选择WebPack版本，则wei完全免费版本，有一些功能的限制，但目前足够做开发发一共是15个G
